# 我对信息素养实践课程的认识  

## 信息素养是什么  
信息素养（Information Literacy）更确切的名称应该是**信息文化（Information Literacy)**
信息素养是一种基本能力,是一种*对信息社会的适应能力*。

## 信息素养的实践课程  
老师不会过分注重学科知识的学习，而关心如何引导学生应用信息技术工具来解决问题，特别是通过把信息技术的学习与学科教学相结合，让学生*把技术作为获取知识和加工信息、为解决问题而服务的工具*。
